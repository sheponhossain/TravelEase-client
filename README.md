## ‚ú® Key Features

## User-Specific Dashboard: A personalized experience where logged-in users can view their specific booking history and transaction records filtered by their unique email address.
## Dynamic Wishlist System: Users can save their favorite vehicles to a personal wishlist. The data is stored securely in MongoDB and is retrieved specifically for each authenticated user.
## Advanced Smart Filtering: A high-performance filtering system that allows users to browse cars by Category (SUV, Luxury, Sedan), Transmission (Manual/Auto), and Fuel Type.
